GoTenna Client
===================

Library for interfacing GoTenna devices over BTLE.

#Getting Started (Gradle / Android Studio)

Add gradle dependency to your application.
```gradle
compile 'com.ToxicBakery.library.btle.gotenna:gotenna-client:+'
```
